General Info

Soul Frames was created using Ionic/Angular over the course of about 6 months. The game is still pretty new and frame measurements are still in progress, for that reason the app does not contain any actual frame data yet! I will add this data as I find contributors and the data becomes available. For the time being, the app can be used for the move lists and extensive filter functionality.
